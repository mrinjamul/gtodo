# gtodo : A simple todo list application

## Build Instructions

Clone repository

`git clone [repo location]`

`cd gtodo`

Install modules

`go mod tidy`

Build

`go build`

Create data files

`touch $HOME/.gtodo.json`

Run ,

`./gtodo`

## INSTALLING

copy into any $PATH

example,

`cp gtodo $GOPATH/bin`
